# Healthcare Worker Burnout Analysis

Analysis of healthcare worker burnout using data from multiple international studies based on the Copenhagen Burnout Inventory (CBI) scales.

Quick start
-----------
1. Create and activate venv:
   python -m venv .venv
   .venv\Scripts\activate  # Windows
2. Install deps:
   pip install -r requirements.txt

Running analysis
----------------
- Open notebooks: jupyter notebook notebooks/
- Run a script: python notebooks/GEDA_script.py

Project structure, data pipeline, development and testing notes are in this repository. See requirements.txt for exact dependency versions.
